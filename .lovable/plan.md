

# План: Преобразование армянских символов в Unicode escape-последовательности

## Цель
Преобразовать все прямые армянские символы в файлах локализации в Unicode escape-последовательности (`\uXXXX`) согласно стандартам проекта.

## Файлы для обработки

| Файл | Строк | Статус |
|------|-------|--------|
| `hy/dashboard.json` | 36 | Требует преобразования |
| `hy/disclaimer.json` | 21 | Требует преобразования |
| `hy/errors.json` | 30 | Требует преобразования |
| `hy/ocr.json` | 28 | Требует преобразования |
| `hy/usage.json` | 23 | Требует преобразования |

## Технические детали

Каждый армянский символ будет заменён на его Unicode escape-последовательность:
- Диапазон армянских символов: `U+0531` - `U+058F`
- Пример: `Վdelays → `\u054E\u0561\u0570\u0561\u0576\u0561\u056F`

## Шаги реализации

1. **dashboard.json** - Преобразовать все 36 строк с армянским текстом
2. **disclaimer.json** - Преобразовать все 21 строку  
3. **errors.json** - Преобразовать все 30 строк
4. **ocr.json** - Преобразовать все 28 строк
5. **usage.json** - Преобразовать все 23 строки

## Примечание
Кириллические символы в русских файлах (`ru/*.json`) остаются без изменений по запросу пользователя.
